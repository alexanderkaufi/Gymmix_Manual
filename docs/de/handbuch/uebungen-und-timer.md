# Übungen und Timer

## Aktive Trainingsansicht

Pfad: Training -> Übung auswählen (z. B. `Dumbbell Fly`)

In der aktiven Ansicht kannst du:

- Sätze erfassen
- Weitere Sätze hinzufügen
- Notizen speichern
- Gewicht und Wiederholungen anpassen
- Die Übung als erledigt markieren

![Aktive Übungsansicht](../../assets/images/figures/active-workout-exercise-view.png)

## Timer

Pfad: Aktive Übung -> Zeit oben in der Mitte antippen

Der Timer hilft dir bei Satzpausen:

- Neuen Timer setzen
- Laufenden Timer stoppen

![Timer-Dialog](../../assets/images/figures/timer-dialog.png)

## Übungsnotizen

Pfad: Aktive Übung -> `Notes`

Notizen helfen dir zum Beispiel bei:

- Bank- oder Sitzposition
- Tempo- und Technikhinweisen
- Tagesform oder Schmerzen

![Übungsnotizen](../../assets/images/figures/exercise-notes.png)

## Detaillierte Satzeinstellungen

Pfad: Aktive Übung -> mittlere Karte (Weight/Reps)

Hier stellst du ein:

- Reps (Plus/Minus)
- Qualität (Sterne)
- Gewicht

![Detaillierte Satzeinstellungen](../../assets/images/figures/detailed-set-settings.png)

## Gewicht und Step Value

Pfad: Satzeinstellungen -> Gewichtswert

Du kannst:

- Das exakte Gewicht eingeben
- Den `Step Value` festlegen, z. B. 1,00 kg pro Klick

![Gewicht anpassen](../../assets/images/figures/step-value-weight-adjustment.png)

![Step Value einstellen](../../assets/images/figures/step-value.png)
