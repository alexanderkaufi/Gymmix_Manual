# Daten und Statistik

## Datenverwaltung

Pfad: Hauptmenü -> `Data Management`

Hier findest du alle Fortschrittsdaten in drei Bereichen:

- `Body Data`
- `Archived Trainings`
- `Statistics`

![Datenverwaltung – Übersicht](../../assets/images/figures/data-management-overview.png)

## Statistik und Analyse

Pfad: `Data Management` -> `Statistics`

Die Diagramme zeigen dir:

- Körperdaten im zeitlichen Verlauf
- Trainingswerte über einen längeren Zeitraum

So erkennst du Trends und langfristige Entwicklungen.

![Statistik und Analyse](../../assets/images/figures/statistics-analysis.png)

## Archivierte Trainings

Pfad: `Data Management` -> `Archived Trainings`

Das Archiv ist dein Trainingstagebuch:

- Frühere Sessions ansehen
- Trainingsblöcke vergleichen
- Motivation über den Verlauf aufbauen

![Archivierte Trainings](../../assets/images/figures/archived-trainings.png)
