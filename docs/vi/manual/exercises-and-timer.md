# Bài tập và hẹn giờ

## Màn hình bài tập đang hoạt động

Đường dẫn: tập luyện -> chọn bài tập (ví dụ `Dumbbell Fly`)

Trong màn hình này, bạn có thể:

- ghi các set,
- thêm set bổ sung,
- lưu ghi chú,
- chỉnh mức tạ và số lần,
- đánh dấu bài tập đã hoàn thành.

![Màn hình bài tập đang hoạt động](../../assets/images/figures/active-workout-exercise-view.png)

## Hẹn giờ

Đường dẫn: bài tập đang hoạt động -> chạm vào thời gian ở giữa phía trên

Bộ hẹn giờ giúp quản lý thời gian nghỉ giữa các set:

- đặt thời gian mới,
- dừng bộ hẹn giờ đang chạy.

![Hộp thoại hẹn giờ](../../assets/images/figures/timer-dialog.png)

## Ghi chú bài tập

Đường dẫn: bài tập đang hoạt động -> `Notes`

Dùng ghi chú cho:

- vị trí ghế/ghế tựa,
- nhắc nhở về nhịp độ và kỹ thuật,
- tình trạng cơ thể hoặc đau mỏi trong ngày.

![Ghi chú bài tập](../../assets/images/figures/exercise-notes.png)

## Cài đặt set chi tiết

Đường dẫn: bài tập đang hoạt động -> thẻ giữa (tạ/số lần)

Bạn có thể cấu hình:

- số lần (plus/minus),
- mức chất lượng (sao),
- mức tạ.

![Cài đặt set chi tiết](../../assets/images/figures/detailed-set-settings.png)

## Mức tạ và Step Value

Đường dẫn: cài đặt set -> giá trị tạ

Bạn có thể:

- nhập chính xác mức tạ,
- đặt `Step Value`, ví dụ 1.00 kg cho mỗi lần bấm.

![Điều chỉnh mức tạ](../../assets/images/figures/step-value-weight-adjustment.png)

![Cài đặt Step Value](../../assets/images/figures/step-value.png)
