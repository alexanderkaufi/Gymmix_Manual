# Cài đặt và gói thuê bao

## Cài đặt

Đường dẫn: menu chính -> `Settings`

Dùng cài đặt để cá nhân hóa Gymmix.

![Tổng quan cài đặt](../../assets/images/figures/settings-overview.png)

## Tùy chỉnh giao diện

Đường dẫn: `Settings` -> `Select Theme`

Chọn chủ đề màu bạn muốn sử dụng.

![Chọn chủ đề](../../assets/images/figures/settings-theme.png)

## Chọn đơn vị đo

Đường dẫn: `Settings` -> `Select Units`

Thiết lập đơn vị đo toàn cục, ví dụ:

- kg hoặc lbs,
- cm hoặc in.

![Cấu hình đơn vị](../../assets/images/figures/settings-units.png)

## Gói thuê bao Gymmix

Đường dẫn: `Settings` -> `Manage subscriptions`

Quản lý thuê bao và mở khóa tính năng pro.

![Quản lý thuê bao](../../assets/images/figures/settings-subscription.png)

Quay lại tổng quan: [Sổ tay Gymmix](./index.md)
