# Кратко о Gymmix

Gymmix — это приложение для тренировок, в котором удобно планировать, фиксировать и измерять прогресс.

Эта страница сделана короткой для быстрого знакомства.

## Что можно делать в Gymmix

- Собирать план тренировок за несколько шагов
- Записывать упражнения, вес и повторения
- Анализировать прогресс по статистике и истории
- Настраивать тему, единицы измерения и подписку

## Старт за 60 секунд

1. Откройте `Training Overview`.
2. Создайте первую тренировку через `Add Training`.
3. Добавьте упражнения через `Add Exercise`.
4. Запустите тренировку и фиксируйте подходы в активном экране.
