# Руководство Gymmix

Добро пожаловать в руководство Gymmix.

Это руководство охватывает ключевые функции и основные сценарии работы.

Обновлено: 2 марта 2026

## Быстрый старт

1. Откройте `Training Overview` в главном меню.
2. Создайте тренировку через `Add Training`.
3. Добавьте упражнения через `Add Exercise`.
4. Записывайте вес, повторения и заметки во время тренировки.
5. Отслеживайте прогресс в `Data Management`.
