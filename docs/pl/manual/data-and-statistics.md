# Dane i statystyki

## Zarządzanie danymi

Ścieżka: menu główne -> `Data Management`

Dane postępu są podzielone na trzy obszary:

- Body Data
- Archived Trainings
- Statistics

![Przegląd zarządzania danymi](../../assets/images/figures/data-management-overview.png)

## Statystyki i analiza

Ścieżka: `Data Management` -> `Statistics`

Wykresy pokazują:

- zmiany parametrów ciała w czasie,
- zmiany wartości treningowych w czasie.

To ułatwia wykrywanie trendów i śledzenie postępu długoterminowego.

![Wykresy statystyk](../../assets/images/figures/statistics-analysis.png)

## Zarchiwizowane treningi

Ścieżka: `Data Management` -> `Archived Trainings`

Archiwum działa jak dziennik treningowy:

- przegląd wcześniejszych sesji,
- porównywanie bloków treningowych,
- śledzenie regularności i motywacji.

![Lista zarchiwizowanych treningów](../../assets/images/figures/archived-trainings.png)
