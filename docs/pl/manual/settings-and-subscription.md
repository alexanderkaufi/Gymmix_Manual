# Ustawienia i subskrypcja

## Ustawienia

Ścieżka: menu główne -> `Settings`

W ustawieniach personalizujesz Gymmix.

![Przegląd ustawień](../../assets/images/figures/settings-overview.png)

## Personalizacja wyglądu

Ścieżka: `Settings` -> `Select Theme`

Wybierz preferowany motyw kolorystyczny.

![Wybór motywu](../../assets/images/figures/settings-theme.png)

## Wybór jednostek

Ścieżka: `Settings` -> `Select Units`

Ustaw globalne jednostki, np.:

- kg lub lbs,
- cm lub in.

![Ustawienia jednostek](../../assets/images/figures/settings-units.png)

## Subskrypcja Gymmix

Ścieżka: `Settings` -> `Manage subscriptions`

Zarządzaj subskrypcją i odblokuj funkcje pro.

![Zarządzanie subskrypcją](../../assets/images/figures/settings-subscription.png)
