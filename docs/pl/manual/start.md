# Start

## Ekran główny

Lokalizacja: menu główne

Ekran główny to centrum Gymmix. Stąd możesz otworzyć:

- Training Overview
- Data Management
- Settings

![Ekran główny](../../assets/images/figures/start-main-menu.png)

## Główne ścieżki w aplikacji

1. `Training Overview`: planowanie i uruchamianie treningów.
2. `Data Management`: historia i postępy.
3. `Settings`: wygląd, jednostki i subskrypcja.

## Zalecane pierwsze kroki

1. Ustaw jednostki i motyw w `Settings`.
2. Utwórz pierwszy trening.
3. Po treningu sprawdź zapisane dane.
