# Exercises and Timer

## Active Exercise View

Path: Training -> select an exercise (for example `Dumbbell Fly`)

In the active view, you can:

- Track sets
- Add additional sets
- Save notes
- Adjust weight and reps
- Mark the exercise as complete

![Active exercise view](../../assets/images/figures/active-workout-exercise-view.png)

## Timer

Path: Active exercise -> tap the time value at the top center

The timer helps you manage rest time between sets:

- Set a new timer
- Stop a running timer

![Timer dialog](../../assets/images/figures/timer-dialog.png)

## Exercise Notes

Path: Active exercise -> `Notes`

Use notes for:

- Bench or seat settings
- Tempo and form reminders
- Daily condition or pain notes

![Exercise notes](../../assets/images/figures/exercise-notes.png)

## Detailed Set Settings

Path: Active exercise -> center card (Weight/Reps)

Configure:

- Reps (plus/minus)
- Quality rating (stars)
- Weight

![Detailed set settings](../../assets/images/figures/detailed-set-settings.png)

## Weight and Step Value

Path: Set settings -> weight value

You can:

- Enter exact weight
- Define `Step Value`, for example 1.00 kg per click

![Weight adjustment](../../assets/images/figures/step-value-weight-adjustment.png)

![Step value settings](../../assets/images/figures/step-value.png)
