# Training

## Training Overview

Path: Main Menu -> `Training Overview`

This screen is where you plan and manage your weekly workouts.

- `Add Training`: Create a new workout.
- `Edit Mode`: Reorder, rename, or delete training days.

![Training overview](../../assets/images/figures/training-overview.png)

## Create a New Workout

Path: `Training Overview` -> `Add Training`

On this screen, you can:

- Set a workout name.
- Add movements from the exercise library.
- Finish and save using `End Training`.

![New training setup](../../assets/images/figures/new-training-setup.png)

## Select from the Exercise Library

Path: Training setup -> `Add Exercise`

The library includes more than 110 exercises. Use:

- Equipment filter (`Sort` icon, top right)
- Multi-level body-part sorting
- `Custom Exercise` for your own entries

![Exercise library](../../assets/images/figures/exercise-library.png)

## Equipment Filter (Sort)

Filter all exercises by available equipment:

- Weights
- Machines
- Body Weight

Extra: Rename the current workout session, for example `Heavy Chest Day`.

![Equipment filter sort](../../assets/images/figures/equipment-filter-sort.png)

## Multi-Level Sorting (Body Parts)

Exercises are grouped by muscle group so you can find relevant movements faster.

![Body-part sorting](../../assets/images/figures/body-part-sorting.png)

## Select a Movement

Select a specific movement from the list, for example `Dumbbell Fly`.

![Select movement](../../assets/images/figures/select-movement.png)

## Add a Custom Exercise

Use `Custom Exercise` when a specific variation is missing from the standard library.

![Custom exercise](../../assets/images/figures/custom-exercise.png)
