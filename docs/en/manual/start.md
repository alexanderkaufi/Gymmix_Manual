# Start

## Home Screen

Location: Main Menu

The home screen is your central Gymmix hub. From here, you can open:

- Training Overview
- Data Management
- Settings

![Home screen in main menu](../../assets/images/figures/start-main-menu.png)

## Main App Paths

1. `Training Overview`: Plan and start workouts.
2. `Data Management`: Review history and progress.
3. `Settings`: Configure design, units, and subscription.

## Recommended First Steps

1. Set units and design in settings.
2. Create your first training session.
3. Review the recorded data right after training.
