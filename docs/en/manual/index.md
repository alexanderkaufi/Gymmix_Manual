# Gymmix Manual

Welcome to the Gymmix Manual.

This guide covers the core features and the most important workflows.

Last updated: March 1, 2026

## Quick Start

1. Open `Training Overview` from the main menu.
2. Create a workout with `Add Training`.
3. Add movements with `Add Exercise`.
4. Track weight, reps, and notes during your workout.
5. Review long-term progress in `Data Management`.
