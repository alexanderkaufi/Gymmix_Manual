# Посібник Gymmix

Ласкаво просимо до посібника Gymmix.

Цей посібник охоплює ключові функції та найважливіші сценарії роботи.

Оновлено: 2 березня 2026

## Швидкий старт

1. Відкрийте `Training Overview` у головному меню.
2. Створіть тренування через `Add Training`.
3. Додайте вправи через `Add Exercise`.
4. Фіксуйте вагу, повторення й нотатки під час тренування.
5. Переглядайте прогрес у `Data Management`.
