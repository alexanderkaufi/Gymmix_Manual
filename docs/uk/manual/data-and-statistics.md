# Дані та статистика

## Керування даними

Шлях: головне меню -> `Data Management`

Усі дані прогресу згруповані у три блоки:

- Body Data
- Archived Trainings
- Statistics

![Огляд керування даними](../../assets/images/figures/data-management-overview.png)

## Статистика та аналіз

Шлях: `Data Management` -> `Statistics`

Графіки показують огляд:

- змін показників тіла з часом,
- змін тренувальних показників з часом.

Це допомагає бачити тенденції та відстежувати довгостроковий прогрес.

![Графіки статистики та аналізу](../../assets/images/figures/statistics-analysis.png)

## Архів тренувань

Шлях: `Data Management` -> `Archived Trainings`

Архів працює як тренувальний журнал:

- перегляд попередніх сесій,
- порівняння тренувальних блоків,
- контроль регулярності та мотивації.

![Список архівних тренувань](../../assets/images/figures/archived-trainings.png)

Повернутися до огляду: [Посібник Gymmix](./index.md)
