# Вправи й таймер

## Екран активної вправи

Шлях: тренування -> вибрати вправу (наприклад `Dumbbell Fly`)

В активному екрані ви можете:

- фіксувати підходи,
- додавати додаткові підходи,
- зберігати нотатки,
- коригувати вагу та повторення,
- позначати вправу як завершену.

![Екран активної вправи](../../assets/images/figures/active-workout-exercise-view.png)

## Таймер

Шлях: активна вправа -> натиснути на значення часу зверху по центру

Таймер допомагає керувати відпочинком між підходами:

- встановити новий таймер,
- зупинити запущений таймер.

![Діалог таймера](../../assets/images/figures/timer-dialog.png)

## Нотатки до вправи

Шлях: активна вправа -> `Notes`

Використовуйте нотатки для:

- налаштувань лави або сидіння,
- нагадувань про темп і техніку,
- записів про самопочуття або біль.

![Нотатки до вправи](../../assets/images/figures/exercise-notes.png)

## Детальні налаштування підходу

Шлях: активна вправа -> центральна картка (вага/повторення)

Налаштуйте:

- повторення (plus/minus),
- оцінку якості (зірки),
- вагу.

![Детальні налаштування підходу](../../assets/images/figures/detailed-set-settings.png)

## Вага та Step Value

Шлях: налаштування підходу -> значення ваги

Ви можете:

- ввести точну вагу,
- задати `Step Value`, наприклад 1.00 кг за натискання.

![Налаштування ваги](../../assets/images/figures/step-value-weight-adjustment.png)

![Налаштування Step Value](../../assets/images/figures/step-value.png)
