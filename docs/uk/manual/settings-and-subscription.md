# Налаштування та підписка

## Налаштування

Шлях: головне меню -> `Settings`

Використовуйте налаштування для персоналізації Gymmix.

![Огляд налаштувань](../../assets/images/figures/settings-overview.png)

## Налаштування дизайну

Шлях: `Settings` -> `Select Theme`

Оберіть бажану кольорову тему.

![Вибір теми](../../assets/images/figures/settings-theme.png)

## Вибір одиниць

Шлях: `Settings` -> `Select Units`

Встановіть глобальні одиниці вимірювання, наприклад:

- kg або lbs,
- cm або in.

![Налаштування одиниць](../../assets/images/figures/settings-units.png)

## Підписка Gymmix

Шлях: `Settings` -> `Manage subscriptions`

Керуйте підпискою та відкривайте pro-функції.

![Керування підпискою](../../assets/images/figures/settings-subscription.png)

Повернутися до огляду: [Посібник Gymmix](./index.md)
