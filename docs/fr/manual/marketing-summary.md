# Résumé Gymmix

Gymmix est votre application d'entraînement focalisée sur la planification, le suivi et la progression mesurable.

Cette page est volontairement courte pour les nouveaux visiteurs.

## Ce que vous pouvez faire avec Gymmix

- Créer un plan d'entraînement en quelques étapes
- Enregistrer exercices, poids et répétitions
- Suivre les progrès via statistiques et historique
- Personnaliser le thème, les unités et l'abonnement

## Commencer en 60 secondes

1. Ouvrez `Training Overview`.
2. Créez votre premier entraînement avec `Add Training`.
3. Ajoutez des exercices via `Add Exercise`.
4. Lancez la séance et enregistrez vos séries dans la vue active.
