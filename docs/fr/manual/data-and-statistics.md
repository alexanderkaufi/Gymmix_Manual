# Données et statistiques

## Gestion des données

Chemin : menu principal -> `Data Management`

Toutes les données de progression sont regroupées en trois zones :

- Body Data
- Archived Trainings
- Statistics

![Vue d'ensemble des données](../../assets/images/figures/data-management-overview.png)

## Statistiques et analyse

Chemin : `Data Management` -> `Statistics`

Les graphiques donnent une vue d'ensemble de :

- l'évolution des métriques corporelles,
- l'évolution des valeurs d'entraînement.

Cela aide à identifier les tendances et suivre la progression long terme.

![Graphiques de statistiques](../../assets/images/figures/statistics-analysis.png)

## Entraînements archivés

Chemin : `Data Management` -> `Archived Trainings`

L'archive agit comme un journal d'entraînement :

- revoir les séances passées,
- comparer des blocs d'entraînement,
- suivre régularité et motivation.

![Liste des entraînements archivés](../../assets/images/figures/archived-trainings.png)
