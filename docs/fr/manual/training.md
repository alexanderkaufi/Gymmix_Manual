# Entraînement

## Vue d'ensemble des entraînements

Chemin : menu principal -> `Training Overview`

Cet écran permet de planifier et gérer les entraînements de la semaine.

- `Add Training` : créer une nouvelle séance.
- `Edit Mode` : réordonner, renommer ou supprimer des jours.

![Vue d'ensemble des entraînements](../../assets/images/figures/training-overview.png)

## Créer une nouvelle séance

Chemin : `Training Overview` -> `Add Training`

Sur cet écran, vous pouvez :

- définir un nom de séance,
- ajouter des mouvements depuis la bibliothèque,
- enregistrer avec `End Training`.

![Nouvelle séance](../../assets/images/figures/new-training-setup.png)

## Sélection depuis la bibliothèque d'exercices

Chemin : configuration séance -> `Add Exercise`

La bibliothèque contient plus de 110 exercices. Utilisez :

- le filtre équipement (`Sort`, en haut à droite),
- le tri par groupes musculaires,
- `Custom Exercise` pour vos propres entrées.

![Bibliothèque d'exercices](../../assets/images/figures/exercise-library.png)

## Filtre équipement

Filtrez les exercices selon le matériel disponible :

- haltères / poids,
- machines,
- poids du corps.

Astuce : vous pouvez renommer la séance en cours, par exemple `Heavy Chest Day`.

![Filtre équipement](../../assets/images/figures/equipment-filter-sort.png)

## Tri par groupes musculaires

Les exercices sont groupés par zone musculaire pour trouver plus vite les mouvements adaptés.

![Tri par groupes musculaires](../../assets/images/figures/body-part-sorting.png)

## Sélectionner un mouvement

Choisissez un mouvement spécifique, par exemple `Dumbbell Fly`.

![Sélection d'un mouvement](../../assets/images/figures/select-movement.png)

## Ajouter un exercice personnalisé

Utilisez `Custom Exercise` lorsqu'une variation manque dans la bibliothèque standard.

![Exercice personnalisé](../../assets/images/figures/custom-exercise.png)
