# Exercices et minuteur

## Vue d'exercice active

Chemin : entraînement -> sélectionner un exercice (exemple `Dumbbell Fly`)

Dans la vue active, vous pouvez :

- suivre les séries,
- ajouter des séries,
- enregistrer des notes,
- ajuster poids et répétitions,
- marquer l'exercice comme terminé.

![Vue d'exercice active](../../assets/images/figures/active-workout-exercise-view.png)

## Minuteur

Chemin : exercice actif -> toucher la valeur de temps en haut au centre

Le minuteur vous aide à gérer la récupération entre les séries :

- définir une nouvelle durée,
- arrêter un minuteur en cours.

![Boîte de dialogue minuteur](../../assets/images/figures/timer-dialog.png)

## Notes d'exercice

Chemin : exercice actif -> `Notes`

Utilisez les notes pour :

- réglages de banc / siège,
- rappels de tempo et technique,
- état du jour ou douleur.

![Notes d'exercice](../../assets/images/figures/exercise-notes.png)

## Réglages détaillés d'une série

Chemin : exercice actif -> carte centrale (poids/répétitions)

Configurez :

- répétitions (plus/moins),
- qualité de la série (étoiles),
- poids.

![Réglages détaillés de série](../../assets/images/figures/detailed-set-settings.png)

## Poids et Step Value

Chemin : réglages de série -> valeur de poids

Vous pouvez :

- entrer un poids précis,
- définir `Step Value`, par exemple 1.00 kg par clic.

![Ajustement du poids](../../assets/images/figures/step-value-weight-adjustment.png)

![Réglage de Step Value](../../assets/images/figures/step-value.png)
