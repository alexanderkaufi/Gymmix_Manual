# Manuel Gymmix

Bienvenue dans le manuel Gymmix.

Ce guide couvre les fonctions principales et les flux les plus importants.

Dernière mise à jour : 2 mars 2026

## Démarrage rapide

1. Ouvrez `Training Overview` depuis le menu principal.
2. Créez un entraînement avec `Add Training`.
3. Ajoutez des exercices avec `Add Exercise`.
4. Enregistrez le poids, les répétitions et les notes pendant la séance.
5. Consultez votre progression dans `Data Management`.
