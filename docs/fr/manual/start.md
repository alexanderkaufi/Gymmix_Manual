# Démarrage

## Écran d'accueil

Emplacement : menu principal

L'écran d'accueil est le point central de Gymmix. Depuis cet écran, vous pouvez ouvrir :

- Training Overview
- Data Management
- Settings

![Écran d'accueil](../../assets/images/figures/start-main-menu.png)

## Chemins principaux de l'app

1. `Training Overview` : planifier et lancer les séances.
2. `Data Management` : consulter l'historique et la progression.
3. `Settings` : configurer le design, les unités et l'abonnement.

## Premières étapes recommandées

1. Réglez les unités et le thème dans `Settings`.
2. Créez votre première séance.
3. Vérifiez les données enregistrées juste après l'entraînement.
