# 设置与订阅

## 设置

路径：主菜单 -> `Settings`

通过设置页面个性化 Gymmix。

![设置总览](../../assets/images/figures/settings-overview.png)

## 自定义外观

路径：`Settings` -> `Select Theme`

选择你喜欢的应用配色主题。

![主题选择](../../assets/images/figures/settings-theme.png)

## 选择单位

路径：`Settings` -> `Select Units`

设置全局单位，例如：

- kg 或 lbs
- cm 或 in

![单位设置](../../assets/images/figures/settings-units.png)

## Gymmix 订阅

路径：`Settings` -> `Manage subscriptions`

管理订阅并解锁专业功能。

![订阅管理](../../assets/images/figures/settings-subscription.png)
