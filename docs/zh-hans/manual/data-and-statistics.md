# 数据与统计

## 数据管理

路径：主菜单 -> `Data Management`

所有进度数据分为三个区域：

- Body Data
- Archived Trainings
- Statistics

![数据管理总览](../../assets/images/figures/data-management-overview.png)

## 统计与分析

路径：`Data Management` -> `Statistics`

图表可查看：

- 身体指标随时间变化，
- 训练指标随时间变化。

这有助于识别趋势并跟踪长期进度。

![统计与分析图表](../../assets/images/figures/statistics-analysis.png)

## 已归档训练

路径：`Data Management` -> `Archived Trainings`

归档相当于你的训练日志：

- 回顾过去训练，
- 比较不同训练周期，
- 跟踪稳定性与动力。

![归档训练列表](../../assets/images/figures/archived-trainings.png)
