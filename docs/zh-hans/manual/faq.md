# 常见问题

## 如何开始第一次训练？

打开 `Training Overview`，点击 `Add Training`，选择动作后开始训练。

## 在哪里修改重量单位？

前往 `Settings` -> `Select Units`。

## 在哪里查看已完成训练？

前往 `Data Management` -> `Archived Trainings`。

## 可以添加自定义动作吗？

可以，在动作选择器中使用 `Custom Exercise`。

## Step Value 是什么？

`Step Value` 定义每次点击 plus/minus 时重量变化的步长。
