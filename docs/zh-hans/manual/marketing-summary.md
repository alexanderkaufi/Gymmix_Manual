# Gymmix 营销简介

Gymmix 是一款专注训练流程的应用，帮助你规划、记录并衡量进步。

本页为首次访问用户提供简短概览。

## 你可以用 Gymmix 做什么

- 用几步完成训练计划
- 在训练中记录重量与次数
- 通过统计和历史训练查看进度
- 自定义主题、单位和订阅设置

## 60 秒开始

1. 打开 `Training Overview`。
2. 用 `Add Training` 创建首个训练。
3. 用 `Add Exercise` 添加动作。
4. 开始训练并在当前动作页面记录组数。
