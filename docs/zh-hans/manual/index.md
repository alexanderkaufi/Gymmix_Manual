# Gymmix 使用手册

欢迎使用 Gymmix 使用手册。

本手册覆盖核心功能和最重要的使用流程。

最后更新：2026年3月2日

## 快速开始

1. 在主菜单打开 `Training Overview`。
2. 通过 `Add Training` 创建训练。
3. 通过 `Add Exercise` 添加动作。
4. 训练时记录重量、次数和备注。
5. 在 `Data Management` 查看长期进度。
