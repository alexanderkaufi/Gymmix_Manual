# 动作与计时器

## 当前动作页面

路径：训练 -> 选择一个动作（例如 `Dumbbell Fly`）

在当前动作页面你可以：

- 记录组数，
- 添加额外组，
- 保存备注，
- 调整重量和次数，
- 标记该动作已完成。

![当前动作页面](../../assets/images/figures/active-workout-exercise-view.png)

## 计时器

路径：当前动作 -> 点击顶部中间的时间值

计时器用于管理组间休息：

- 设置新的计时，
- 停止正在运行的计时。

![计时器弹窗](../../assets/images/figures/timer-dialog.png)

## 动作备注

路径：当前动作 -> `Notes`

备注可用于：

- 卧推凳/座椅设置，
- 节奏与动作技术提醒，
- 当天状态或疼痛记录。

![动作备注](../../assets/images/figures/exercise-notes.png)

## 详细组设置

路径：当前动作 -> 中央卡片（重量/次数）

可配置：

- 次数（plus/minus），
- 质量评分（星级），
- 重量。

![详细组设置](../../assets/images/figures/detailed-set-settings.png)

## 重量与 Step Value

路径：组设置 -> 重量值

你可以：

- 输入精确重量，
- 设置 `Step Value`，例如每次点击 1.00 kg。

![重量调整](../../assets/images/figures/step-value-weight-adjustment.png)

![Step Value 设置](../../assets/images/figures/step-value.png)
