# Configuración y suscripción

## Configuración

Ruta: Menú principal -> `Settings`

Usa la configuración para personalizar Gymmix.

![Vista general de configuración](../../assets/images/figures/settings-overview.png)

## Personalizar diseño

Ruta: `Settings` -> `Select Theme`

Elige el tema de color que prefieras.

![Selección de tema](../../assets/images/figures/settings-theme.png)

## Seleccionar unidades

Ruta: `Settings` -> `Select Units`

Define las unidades globales, por ejemplo:

- kg o lbs
- cm o in

![Configuración de unidades](../../assets/images/figures/settings-units.png)

## Suscripción de Gymmix

Ruta: `Settings` -> `Manage subscriptions`

Gestiona tu suscripción y desbloquea funciones pro.

![Gestión de suscripción](../../assets/images/figures/settings-subscription.png)
