# Datos y estadísticas

## Gestión de datos

Ruta: Menú principal -> `Data Management`

Todos los datos de progreso se agrupan en tres áreas:

- Body Data
- Archived Trainings
- Statistics

![Vista general de gestión de datos](../../assets/images/figures/data-management-overview.png)

## Estadísticas y análisis

Ruta: `Data Management` -> `Statistics`

Los gráficos te ofrecen una vista general de:

- Métricas corporales a lo largo del tiempo
- Valores de entrenamiento a lo largo del tiempo

Así puedes identificar tendencias y seguir tu progreso a largo plazo.

![Gráficos de estadísticas y análisis](../../assets/images/figures/statistics-analysis.png)

## Entrenamientos archivados

Ruta: `Data Management` -> `Archived Trainings`

El archivo funciona como diario de entrenamiento:

- Revisar sesiones anteriores
- Comparar bloques de entrenamiento
- Seguir constancia y motivación

![Lista de entrenamientos archivados](../../assets/images/figures/archived-trainings.png)
