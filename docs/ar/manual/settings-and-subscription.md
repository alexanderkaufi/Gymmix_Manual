# الإعدادات والاشتراك

## الإعدادات

المسار: القائمة الرئيسية -> `Settings`

استخدم الإعدادات لتخصيص Gymmix.

![نظرة عامة على الإعدادات](../../assets/images/figures/settings-overview.png)

## تخصيص التصميم

المسار: `Settings` -> `Select Theme`

اختر سمة الألوان التي تفضلها.

![اختيار السمة](../../assets/images/figures/settings-theme.png)

## اختيار الوحدات

المسار: `Settings` -> `Select Units`

حدد وحدات القياس العامة، مثل:

- kg أو lbs
- cm أو in

![إعداد الوحدات](../../assets/images/figures/settings-units.png)

## اشتراك Gymmix

المسار: `Settings` -> `Manage subscriptions`

قم بإدارة الاشتراك وفتح المزايا الاحترافية.

![إدارة الاشتراك](../../assets/images/figures/settings-subscription.png)

العودة إلى النظرة العامة: [دليل Gymmix](./index.md)
