# التمارين والمؤقت

## شاشة التمرين النشط

المسار: التدريب -> اختيار تمرين (مثل `Dumbbell Fly`)

في الشاشة النشطة يمكنك:

- تسجيل الجولات
- إضافة جولات إضافية
- حفظ الملاحظات
- تعديل الوزن والتكرارات
- وضع علامة اكتمال للتمرين

![شاشة التمرين النشط](../../assets/images/figures/active-workout-exercise-view.png)

## المؤقت

المسار: التمرين النشط -> الضغط على قيمة الوقت في أعلى الوسط

يساعدك المؤقت على إدارة وقت الراحة بين الجولات:

- تعيين مؤقت جديد
- إيقاف المؤقت الجاري

![نافذة المؤقت](../../assets/images/figures/timer-dialog.png)

## ملاحظات التمرين

المسار: التمرين النشط -> `Notes`

استخدم الملاحظات من أجل:

- إعدادات المقعد أو البنش
- تذكير بالإيقاع والتكنيك
- ملاحظات الحالة اليومية أو الألم

![ملاحظات التمرين](../../assets/images/figures/exercise-notes.png)

## إعدادات الجولة التفصيلية

المسار: التمرين النشط -> البطاقة الوسطى (وزن/تكرارات)

يمكنك ضبط:

- التكرارات (plus/minus)
- تقييم الجودة (نجوم)
- الوزن

![إعدادات الجولة التفصيلية](../../assets/images/figures/detailed-set-settings.png)

## الوزن و Step Value

المسار: إعدادات الجولة -> قيمة الوزن

يمكنك:

- إدخال الوزن بدقة
- تحديد `Step Value` مثل 1.00 كجم لكل ضغطة

![تعديل الوزن](../../assets/images/figures/step-value-weight-adjustment.png)

![إعداد Step Value](../../assets/images/figures/step-value.png)
