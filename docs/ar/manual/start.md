# البدء

## الشاشة الرئيسية

الموقع: القائمة الرئيسية

الشاشة الرئيسية هي مركز التحكم في Gymmix. من هنا يمكنك فتح:

- Training Overview
- Data Management
- Settings

![الشاشة الرئيسية](../../assets/images/figures/start-main-menu.png)

## المسارات الرئيسية داخل التطبيق

1. `Training Overview`: تخطيط وبدء التمارين.
2. `Data Management`: مراجعة السجل والتقدم.
3. `Settings`: تخصيص التصميم والوحدات والاشتراك.

## خطوات أولى مقترحة

1. اضبط الوحدات والتصميم داخل `Settings`.
2. أنشئ أول حصة تدريب.
3. راجع البيانات المسجلة مباشرة بعد التدريب.

العودة إلى النظرة العامة: [دليل Gymmix](./index.md)
