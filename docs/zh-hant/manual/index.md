# Gymmix 使用手冊

歡迎使用 Gymmix 使用手冊。

本手冊涵蓋核心功能與最重要的使用流程。

最後更新：2026年3月2日

## 快速開始

1. 在主選單開啟 `Training Overview`。
2. 透過 `Add Training` 建立訓練。
3. 透過 `Add Exercise` 新增動作。
4. 訓練中記錄重量、次數與備註。
5. 在 `Data Management` 查看長期進度。
