# 資料與統計

## 資料管理

路徑：主選單 -> `Data Management`

所有進度資料分成三個區塊：

- Body Data
- Archived Trainings
- Statistics

![資料管理總覽](../../assets/images/figures/data-management-overview.png)

## 統計與分析

路徑：`Data Management` -> `Statistics`

圖表可查看：

- 身體指標隨時間變化，
- 訓練指標隨時間變化。

這能幫助你辨識趨勢並追蹤長期進度。

![統計與分析圖表](../../assets/images/figures/statistics-analysis.png)

## 已封存訓練

路徑：`Data Management` -> `Archived Trainings`

封存區就像你的訓練日誌：

- 回顧過去訓練，
- 比較不同訓練週期，
- 追蹤穩定度與動力。

![封存訓練清單](../../assets/images/figures/archived-trainings.png)
