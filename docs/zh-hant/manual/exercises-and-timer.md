# 動作與計時器

## 目前動作畫面

路徑：訓練 -> 選擇一個動作（例如 `Dumbbell Fly`）

在目前動作畫面你可以：

- 記錄組數，
- 新增額外組，
- 儲存備註，
- 調整重量與次數，
- 標記該動作已完成。

![目前動作畫面](../../assets/images/figures/active-workout-exercise-view.png)

## 計時器

路徑：目前動作 -> 點擊頂部中央的時間值

計時器可協助管理組間休息：

- 設定新的計時，
- 停止正在執行的計時。

![計時器對話框](../../assets/images/figures/timer-dialog.png)

## 動作備註

路徑：目前動作 -> `Notes`

備註可用於：

- 臥推椅/座椅設定，
- 節奏與動作品質提醒，
- 當日狀態或疼痛紀錄。

![動作備註](../../assets/images/figures/exercise-notes.png)

## 詳細組設定

路徑：目前動作 -> 中央卡片（重量/次數）

可設定：

- 次數（plus/minus），
- 品質評分（星等），
- 重量。

![詳細組設定](../../assets/images/figures/detailed-set-settings.png)

## 重量與 Step Value

路徑：組設定 -> 重量值

你可以：

- 輸入精確重量，
- 設定 `Step Value`，例如每次點擊 1.00 kg。

![重量調整](../../assets/images/figures/step-value-weight-adjustment.png)

![Step Value 設定](../../assets/images/figures/step-value.png)
