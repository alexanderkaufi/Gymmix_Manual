# 訓練

## 訓練總覽

路徑：主選單 -> `Training Overview`

此頁面用於規劃與管理每週訓練。

- `Add Training`：建立新訓練。
- `Edit Mode`：調整順序、重新命名或刪除訓練日。

![訓練總覽](../../assets/images/figures/training-overview.png)

## 建立新訓練

路徑：`Training Overview` -> `Add Training`

在此頁面你可以：

- 設定訓練名稱，
- 從動作庫加入動作，
- 使用 `End Training` 完成並儲存。

![新訓練設定](../../assets/images/figures/new-training-setup.png)

## 從動作庫選擇

路徑：訓練設定 -> `Add Exercise`

動作庫包含 110+ 動作，可使用：

- 器材篩選（右上角 `Sort`），
- 依肌群分層排序，
- `Custom Exercise` 加入自訂動作。

![動作庫](../../assets/images/figures/exercise-library.png)

## 器材篩選

依可用器材篩選動作：

- 重量器材
- 固定器材
- 自重

補充：你也可以重新命名當前訓練，例如 `Heavy Chest Day`。

![器材篩選](../../assets/images/figures/equipment-filter-sort.png)

## 依肌群分層排序

動作依肌群分組，能更快找到合適動作。

![肌群排序](../../assets/images/figures/body-part-sorting.png)

## 選擇動作

從清單中選擇特定動作，例如 `Dumbbell Fly`。

![選擇動作](../../assets/images/figures/select-movement.png)

## 新增自訂動作

當標準動作庫缺少你需要的變化時，使用 `Custom Exercise`。

![自訂動作](../../assets/images/figures/custom-exercise.png)
