# Gymmix 行銷簡介

Gymmix 是一款專注訓練流程的應用，幫助你規劃、記錄並衡量進步。

本頁為首次造訪的使用者提供精簡概覽。

## 你可以用 Gymmix 做什麼

- 用幾個步驟建立訓練計畫
- 在訓練中記錄重量與次數
- 透過統計與歷史訓練查看進度
- 自訂主題、單位與訂閱設定

## 60 秒開始

1. 開啟 `Training Overview`。
2. 用 `Add Training` 建立第一個訓練。
3. 用 `Add Exercise` 新增動作。
4. 開始訓練並在當前動作畫面記錄組數。
