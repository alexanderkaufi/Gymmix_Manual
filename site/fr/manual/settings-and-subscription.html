Réglages et abonnement

Réglages

Chemin : menu principal -> Settings

Utilisez les réglages pour personnaliser Gymmix.

Personnaliser le design

Chemin : Settings -> Select Theme

Choisissez votre thème de couleur préféré.

Choisir les unités

Chemin : Settings -> Select Units

Définissez les unités globales, par exemple :

  • kg ou lbs
  • cm ou in

Abonnement Gymmix

Chemin : Settings -> Manage subscriptions

Gérez votre abonnement et débloquez les fonctions pro.
