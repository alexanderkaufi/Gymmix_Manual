Тренировки

Обзор тренировок

Путь: главное меню -> Training Overview

На этом экране вы планируете и управляете недельными тренировками.

  • Add Training: создать новую тренировку.
  • Edit Mode: менять порядок, имя или удалять тренировочные дни.

Создать новую тренировку

Путь: Training Overview -> Add Training

На этом экране можно:

  • задать название тренировки,
  • добавить движения из библиотеки,
  • завершить и сохранить через End Training.

Выбор из библиотеки упражнений

Путь: настройка тренировки -> Add Exercise

Библиотека содержит более 110 упражнений. Используйте:

  • фильтр оборудования (Sort, справа сверху),
  • сортировку по группам мышц,
  • Custom Exercise для собственных упражнений.

Фильтр по оборудованию

Фильтруйте упражнения по доступному оборудованию:

  • веса,
  • тренажеры,
  • собственный вес.

Дополнительно можно переименовать текущую сессию, например Heavy Chest Day.

Сортировка по группам мышц

Упражнения сгруппированы по мышечным группам, чтобы быстрее находить нужные движения.

Выбор движения

Выберите конкретное движение из списка, например Dumbbell Fly.

Добавить пользовательское упражнение

Используйте Custom Exercise, если в стандартной библиотеке нет нужного варианта.
