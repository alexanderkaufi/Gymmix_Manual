設定與訂閱

設定

路徑：主選單 -> Settings

透過設定頁面個人化 Gymmix。

自訂外觀

路徑：Settings -> Select Theme

選擇你偏好的應用色彩主題。

選擇單位

路徑：Settings -> Select Units

設定全域單位，例如：

  • kg 或 lbs
  • cm 或 in

Gymmix 訂閱

路徑：Settings -> Manage subscriptions

管理訂閱並解鎖進階功能。
